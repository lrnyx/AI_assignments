Turing Test and CAPTCHA
This program shows two ideas used in AI to tell whether something is a human or a machine.

Turing Test

The Turing Test was proposed by Alan Turing. The idea is that a judge talks to two participants – one human and one machine. The judge does not know which one is which. After reading their answers, the judge tries to guess which one is the machine.
In this program, the user gives the human response and the program generates a response for the AI. The answers are shown in random order and the judge tries to guess which one is the AI.

CAPTCHA

CAPTCHA is used on many websites to check if the user is a human. It shows some random letters and numbers in an image.
The user has to type the characters shown in the image. If the text matches, the user is verified as a human.
In this program, Python is used to generate a random string and create an image with some noise lines so that bots cannot easily read it.
